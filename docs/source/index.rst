Infinity Tomographic Reconstruction
==============================================


The **Infinity Tomographic Reconstruction** is a Python package that provides a set of tools for parametric 3D tomographic reconstruction mainly listmode-based.

Systems Types
------------
- PET
- CT (in test)
- SPECT
- Compton (in test

Geometry Types
--------------
- Dynamic
   easyPET based
   LinearPET based
- Static
   cylindrical
   polygonal
   planar
